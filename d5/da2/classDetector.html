Detailed Description

The Detector class defines the basic properties of a detector model.

The Detector class collects together a list (vector) of cylindrical detector layers. It is labelled with an identifier and receives a default integrated luminosity. Both of these properties can be read, as well as the detector's acceptance. This is the 'macroscopic' object that is tested in the simulation. Detector models are stored in the src/Detectors/ folder.

Constructor of the Detector class.

The constructor of the Detector class is straightforward. Indeed, depending on the geometry, there is no unique strategy to build the detector out of cylindrical objects. The input thus consists of the identifier (string), the default integrated luminosity (double) and a list of cyl. detector layers.

JSON Pointer

The library supports JSON Pointer (RFC 6901) as alternative means to address structured values.

// a JSON value
json j_original = R"({
"baz": ["one", "two", "three"],
"foo": "bar"
})"_json;
// access members with a JSON pointer (RFC 6901)
j_original["/baz/1"_json_pointer];
// "two"
